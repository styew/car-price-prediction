# Used Car Price Prediction

This project explores a dataset of used and new car sales and builds a Linear Regression model to predict car prices.

## Steps
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering (in progress)
- Linear Regression model (baseline)

## Tools
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## Status
Work in progress (ML model still being improved)

ref:https://www.kaggle.com/datasets/deepcontractor/car-price-prediction-challenge/data
